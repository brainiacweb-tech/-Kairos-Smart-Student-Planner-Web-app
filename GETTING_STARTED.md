# 🚀 Kairos - Quick Start Guide

## Welcome! 👋

Kairos is a modern, responsive web application for managing student assignments, planning schedules, and tracking academic progress—completely in your browser.

---

## ⚡ Quick Start (30 seconds)

### Option 1: Direct Browser Open
1. Navigate to the project folder
2. Double-click `index.html`
3. Click "Continue as Guest" to start immediately

### Option 2: Local Server (Recommended)
```bash
# Using Python 3
python -m http.server 8000

# OR using Node.js
npx http-server

# Then visit: http://localhost:8000
```

## 💾 Your Data

### How It's Stored
All your data stays **on your device** using browser storage:

## 🆘 Troubleshooting

### Data Not Saving?
- **Clear Cache**: Browser > Settings > Clear browsing data
- **Check Storage**: Some browsers limit storage in private mode
- **Try Again**: Reload the page (Ctrl+R or Cmd+R)

### Can't Login?
- **Check Email**: Make sure email is correct
- **Guest Access**: Use "Continue as Guest" to bypass
- **Clear History**: May have conflicting login data

### Page Won't Load?
- **Refresh**: Ctrl+R (Cmd+R on Mac)
- **Check Internet**: Required for first load
- **Try Later**: Server might be busy

### Missing Features?
- **Enable JavaScript**: Required for all features
- **Update Browser**: Use latest version
- **Try Different Browser**: Rule out browser issues
