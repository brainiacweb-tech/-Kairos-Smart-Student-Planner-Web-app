# Quick Start Guide: Lecture Tools Features

## 🎯 What's New

Kairos now has powerful lecture management tools including:
- 📅 Export your schedule to calendar apps
- 📊 Analyze your schedule for conflicts and workload
- 💾 Import/Export as CSV or PDF
- ⏰ Find available study time

---

## 🚀 Getting Started

### 1. **Export Your Schedule to Google Calendar**

**Steps:**
1. Open **Lecture Tools** page
2. Scroll to **Export Schedule** section
3. Click **📅 Export to Calendar (ICS)**
4. File `lectures.ics` downloads
5. Open Google Calendar
6. Click **+** → **Import & Add**
7. Select downloaded `lectures.ics`
8. Done! Schedule appears in calendar

**Benefits:**
- Get calendar reminders
- See lectures alongside other events
- Sync across devices

---

### 2. **Check for Schedule Conflicts**

**Steps:**
1. Open **Lecture Tools** page
2. Find **Analyze Schedule** section
3. Click **🔍 Check for Conflicts**
4. View any overlapping classes

**What it checks:**
- Classes scheduled at same time
- Room double bookings
- Time overlaps

**What to do:**
- If conflicts found → Contact department to reschedule
- If none found → Schedule is clean! ✓

---

### 3. **Understand Your Workload**

**Steps:**
1. Open **Lecture Tools** page
2. Click **📊 View Statistics**
3. See breakdown of:
   - Total classes per week
   - Hours of classes
   - Busiest day
   - Average class duration
   - Daily breakdown

**Example output might show:**
```
📊 Schedule Overview
━━━━━━━━━━━━━━━━━━
Total Weekly Hours: 25.5
Busiest Day: Monday (4 classes)
Average per class: 90 minutes
Total Courses: 5
```

---

### 4. **Find Time to Study**

**Steps:**
1. Open **Lecture Tools** page
2. Click **⏰ Find Available Study Time**
3. See free slots by day
4. Filter by duration (30 min, 60 min, 90 min, etc.)

**Example output:**
```
Monday:
  9:00 - 10:00  (60 minutes free)
  12:00 - 14:00 (120 minutes free)
  
Tuesday:
  10:00 - 12:00 (120 minutes free)
  14:30 - 16:00 (90 minutes free)
```

**Use this to:**
- Schedule study groups
- Plan work sessions
- Avoid time conflicts when signing up for things

---

### 5. **Backup Your Schedule**

**Export Options:**
- **ICS Format** - Use in calendars
- **CSV Format** - Edit in Excel/Sheets
- **PDF Format** - Print or share
- **JSON Format** - For technical users

**Steps to backup:**
1. Open **Lecture Tools**
2. Choose export format
3. File automatically downloads
4. Save to cloud (Google Drive, OneDrive, etc.)

---

### 6. **Share Your Schedule**

**Ways to share:**
1. **Export to CSV** → Share spreadsheet with classmates
2. **Export to PDF** → Email or print
3. **Export to ICS** → Share calendar file

**Steps:**
1. Export in desired format
2. Share file (email, cloud link, etc.)
3. Others can import and view

---

### 7. **Import Schedule from Files**

**If you have existing CSV file:**

**Steps:**
1. Open **Lecture Tools**
2. Find **Import Schedule** section
3. Click **Upload CSV File**
4. Select your `.csv` file
5. Review lectures to be added
6. Click **Confirm Import**

**CSV Format:**
```
Course Name,Day of Week,Start Time,End Time,Room,Instructor,Recurring,Notes
Math 101,Monday,09:00,10:30,Room 101,Prof. Smith,Yes,Lecture 1
Physics 101,Wednesday,14:00,15:30,Lab A,Dr. Johnson,Yes,Lab
```

1. View Statistics regularly

## ❓ Frequently Asked Questions

**Q: Can I import my institution's schedule?**
A: Yes! If it's in CSV format with: Course Name, Day, Start Time, End Time, Room, Instructor

**Q: Will my calendar app show all events correctly?**
A: Yes! Weekly recurring events will appear. Cancel/adjust in calendar app as needed.

**Q: What if I have conflicts?**
A: Conflicts might indicate errors in your data or actual schedule problems. Contact your department.

**Q: Can I edit after importing?**
A: Yes! You can edit individual classes after import in Kairos.

**Q: How often should I backup?**
A: At least once per semester, or after major schedule changes.

**Q: Can I share my schedule with my study group?**
A: Yes! Export as ICS and share the file, or export as CSV to find common free times.

## 🔧 Troubleshooting

**Problem:** Export doesn't download
- **Solution:** 
  1. Check browser download settings
  2. Try different browser
  3. Clear cache and try again

**Problem:** Import fails
- **Solution:**
  1. Verify CSV column headers match exactly
  2. Check day names (Monday, Mon, etc.)
  3. Ensure times are in HH:MM format
  4. Try with smaller test file first

**Problem:** Conflicts showing but shouldn't be
- **Solution:**
  1. Verify all times entered correctly
  2. Check time zones match
  3. Double-check duration calculations

**Problem:** Available slots showing only long gaps
- **Solution:**
  1. Adjust duration filter lower (60 min instead of 120 min)
  2. Add more lectures if very empty schedule
